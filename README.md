# heficience-scripts
Tous les scripts d'installation pour les solutions de l'équipe Heficience.

### Pour les developpeurs
Vous pouvez ajouter/modifier/activer/désactiver/supprimer votre script a l'aide de la commande :
```python
python3 generate.py
```
